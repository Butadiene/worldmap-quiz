# Natural Earth 既定 (de facto) と日本視点 (Japan POV) の差分リスト

`tools/make-jpn-pov.cjs` の開発時 (2026-07, v43) に、Natural Earth 公式の
`ne_10m_admin_0_countries` (既定 = 実効支配ベース) と `ne_10m_admin_0_countries_jpn`
(日本視点。POV 系列は 10m スケールのみ配布) を point-in-polygon で突き合わせて
確認した内容の記録。アプリ利用者向けの文書ではなく開発メモ (アプリ UI には
「Natural Earth の日本視点データに合わせて調整」以上の詳細は書かない方針)。

## 帰属が異なる箇所 (公式 10m データで確認)

| 地域 | 既定 (de facto) | 日本視点 (jpn POV) | アプリ (50m/110m) への反映 |
|---|---|---|---|
| 北方領土 (択捉・国後・色丹・歯舞) | ロシア | 日本 | ✅ 50m で 6 ポリゴン移動 (110m はポリゴン自体なし) |
| 竹島 | 韓国 | 日本 | ―  50m/110m にポリゴンがなく描画されない (帰属表示も発生しない) |
| クリミア半島 | ロシア | ウクライナ | ✅ 50m/110m とも反映 (本土との境界線も解消) |
| ソマリランド | 独立フィーチャ (ISO id なし) | ソマリアに統合 | ✅ 50m/110m とも反映 |
| 北キプロス | 独立フィーチャ (ISO id なし) | キプロスに統合 | ✅ 50m/110m とも反映 |
| シアチェン氷河 | 独立フィーチャ (ISO id なし) | インドに統合 | ✅ 50m で反映 (110m はフィーチャ自体なし) |
| 西サハラ西部 (「砂の壁」以西の実効支配地域) | モロッコ | 西サハラ | ❌ 未反映 — 下記「反映できない差分」参照 |
| ゴラン高原 | イスラエル | **どの国にも帰属しない空白地帯** | ❌ 未反映 — 同上 |

## 差がないことを確認した箇所

- 尖閣諸島 — **両視点とも日本** (既定でも実効支配ベースで日本。50m/110m には
  ポリゴンがないため描画はされない)
- 北千島 (幌筵島など) — 両視点ともロシア (日本の主張は南千島 4 島のみ)
- コソボ — 両視点とも独立フィーチャ (ISO 数値 id なし → アプリでは inert land)
- 台湾・パレスチナ — 両視点とも独立フィーチャ (ISO id あり)
- アクサイチン — 両視点とも中国 / アルナーチャル・プラデーシュ — 両視点ともインド
- 南沙諸島 (スプラトリー) — 両視点とも「Spratly Is.」独立フィーチャ
- フォークランド諸島 — 両視点とも英領

## 反映できない差分 (既知の制限)

**西サハラ西部**と**ゴラン高原**は、world-atlas 50m/110m ではモロッコ本土・
イスラエル本土のポリゴンの内部に含まれており、境界線そのものがポリゴンの中を
通る。make-jpn-pov.cjs はアーク参照の付け替え (= ポリゴン単位の帰属変更) しか
行わないため、この 2 箇所は**デファクト表示のまま**になる。反映するには 10m POV の
境界線でポリゴンを切断するトポロジー編集が必要で、「座標・トポロジー不変」の
方針を崩すため見送り。なおゴラン高原は公式日本視点でも「シリアに返す」のではなく
無帰属の空白として扱われている (2026-07 時点)。

また、ポリゴン内部を通る境界線の**線形**そのもの (例: カシミールの停戦ライン等)
は world-atlas の描線に従っており、10m POV の描線とは一致しない。

## 検証方法

- 各係争地の代表点で point-in-polygon (含有判定 + 極小島は近傍リング探索)。
- Natural Earth のデータは nvkelso/natural-earth-vector リポジトリ master
  (2026-07 取得) の geojson/。NE がデータを更新した場合はこの表も変わりうるので、
  再生成時は make-jpn-pov.cjs のログ (移動一覧) と突き合わせること。
